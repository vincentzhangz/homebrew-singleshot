# Homebrew Tap for Singleshot

This is the official Homebrew tap for [Singleshot](https://github.com/vincentzhangz/singleshot).

## Installation

First, add the tap:

```bash
brew tap vincentzhangz/singleshot
```

Then install `singleshot`:

```bash
brew install singleshot
```

## Updates

The tap is automatically updated to track the latest stable releases of Singleshot. Running `brew upgrade` will fetch the latest version.
